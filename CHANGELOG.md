# CivitasGrid-TLP (TransLuxPop) Changelog

## v1.1.0
Added **Dataset 4**, providing a curated multi-tier dataset with  
**4,293 (MQ)** or **2,833 (HQ)** 30–60 arcsec grids, including:
- transportation network features (OSM-derived)
- nighttime light (VIIRS)
- population (WorldPop)

## v1.0.0
First public release of CivitasGrid-TLP (TransLuxPop), providing  
**873 (Raw)** or **853 (HQ)** 30–60 arcsec grids, including:
- transportation network features (OSM-derived)
- nighttime light (VIIRS)
- population (WorldPop)
